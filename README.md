# NikkiAPI
一个基于FASTAPI框架开发的闪耀暖暖题库API
***
# 部署方法
+ 第一步

  `pip install -r requirements.txt`
+ 第二步

  `python app.py`
  
> 需要Python3.6及以上版本

> 默认运行在127.0.0.1:8000，如需修改，请修改app.py中的`uvicorn.run(app, host="127.0.0.1", port=8000)`
